                     Eigen::Vector2d p3,
                     Eigen::Vector2d closest);

Calculate closest point given waypoint line between p1, p2 and robot position

  • [in] p1 Waypoint 1
  • [in] p2 Waypoint 2
  • [in] p3 Robot position
  • [out] closest Closest point

Returns Unit number denoting where the closest point is on waypoint line. For example, a return value of 0.5 denotes the closest point is half-way (50%) of the waypoint line, alternatively a negative number denotes the closest point is behind the first waypoint.
