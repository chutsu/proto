Trace

\text{tr}(\Mat{A}) = \sum_{i} \Mat{A}_{ii} The trace of a matrix $A$ is simply the sum of all of its diagonal.

Rank

The rank $\rho(\Mat{A})$ of a matrix $\Mat{A}$ of $n$ rows and $m$ columns is defined as the number of independent rows or columns.
  • Full rank (non-singular): $\rho(\Mat{A}) = \min(n, m)$
  • Not full rank (singular): $\rho(\Mat{A}) < \min(n, m)$
